1) 구성파일 작성

  • git 저장소 생성
    git init myinfra && cd myinfra
  • 구성파일 작성
    vi main.tf

  • Terraform 초기화

    terraform init

  • 계획 배포를 하기전 시뮬레이션

    terraform plan

  • 적용 인프라를 배포

    terraform apply
